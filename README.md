# CodeAlpha URL Shortener

This is a simple URL Shortener backend project built using Flask.

## Project Overview

The project allows users to submit a long URL and receive a shortened URL. When the shortened URL is visited, the user will be redirected to the original long URL.

## Tech Stack

- Python
- Flask
- SQLite
- Postman

Foundation setup completed. URL shortener features will be implemented next.
